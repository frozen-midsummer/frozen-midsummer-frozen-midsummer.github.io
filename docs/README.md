## 前言

采用 VuePress2.0.0 + Github Pages 搭建的个人笔记网站，以在不同的地点在线编辑/查阅笔记。

## 快速开始

```bash
##  下载代码
git clone
##  安装依赖（若没安装 pnpm 先 npm install -g pnpm）
pnpm install
```
